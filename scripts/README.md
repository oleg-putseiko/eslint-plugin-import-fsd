# Project Scripts

This directory contains automation scripts for project setup, maintenance, and environment management.

## Workspace & Environment

- **`deps/audit.sh`**: Identifies unstable 0.x.x dependencies and configures Dependabot to prevent breaking updates.

## Internal Utilities

> ⚠️ These scripts are intended for internal coordination and should not be run standalone

- **`utils/log.sh`**: A universal logging utility providing standardized formatting, indentation, and status reporting.
- **`utils/colors.sh`**: A centralized collection of ANSI color codes for consistent visual output across all scripts.
